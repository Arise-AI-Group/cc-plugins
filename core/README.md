# core

Shared skills and utilities for cc-plugins - setup, configuration, and common workflows.

## Installation

```bash
/plugin install core@cc-plugins
```

## Skills

This plugin includes:
- **setup** - Interactive onboarding for CC-Plugins. Detects installed plugins and configures required API credentials.

## Usage

```
/core:skills
```

Or say "set up cc-plugins" to trigger the setup skill.
